# Customer Churn Prediction – Data Science Mini Project

## Project Overview

This project focuses on predicting customer churn in a telecom company using machine learning techniques. Customer churn refers to when a customer stops using the company’s services.

The goal of this project is to analyze customer data and build a model that can predict whether a customer is likely to leave or not.

---

## Objective

The main objectives of this project are:

* To understand customer behavior
* To perform data cleaning and preprocessing
* To analyze patterns using exploratory data analysis (EDA)
* To build a machine learning model for churn prediction
* To evaluate model performance

---

## Dataset Information

The dataset contains telecom customer information such as:

* Tenure (duration of service)
* Monthly Charges
* Contract Type
* Internet Service
* Payment Method
* Other service-related features
* Churn (Target Variable)

The target variable **Churn** indicates whether a customer has left the service (Yes/No).

---

## Machine Learning Model Used

### Decision Tree Classifier

A Decision Tree model was used to classify whether a customer will churn or not.
This model was chosen because it is easy to understand and works well for classification problems.

---

## Model Evaluation

The model was evaluated using:

* Accuracy Score
* Confusion Matrix

The model achieved an accuracy of approximately **79%**, indicating good performance.

---

## Key Insights

* Customers with shorter tenure are more likely to churn
* Customers with higher monthly charges tend to leave more often
* Service-related features also influence customer retention

## Conclusion

The Decision Tree model successfully predicted customer churn with good accuracy.
The analysis shows that customer tenure, monthly charges, and service usage are important factors affecting churn.

These insights can help telecom companies take proactive steps to reduce customer loss and improve customer retention.
